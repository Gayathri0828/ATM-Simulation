# 🏧 ATM Simulation

A simple **ATM Simulation** project built using **Python**. This project demonstrates basic ATM operations through a command-line interface, allowing users to perform banking transactions securely.

## ✨ Features

- 🔐 PIN Verification
- 💰 Check Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- ⚠️ Insufficient Balance Handling
- 📋 Menu-Driven Interface
- 🔄 Continuous Transactions using `while` loop
- 🚪 Exit Option

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code (VS Code)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Gayathri0828/ATM-Simulation.git
```

2. Open the project folder.

3. Run the Python file:

```bash
python atm.py
```
